# hmcts-task

A simple task management app created for a coding challenge

## Running the Applications

### Backend (Spring Boot)

1. Open a terminal and navigate to the `backend` directory:
   ```sh
   cd backend
   ```
2. Start the backend server using Maven Wrapper:
   ```sh
   ./mvnw spring-boot:run
   ```
   On Windows, use:
   ```sh
   mvnw.cmd spring-boot:run
   ```
3. The backend will start on [http://localhost:8080](http://localhost:8080)

### Frontend (React)

1. Open a new terminal and navigate to the `frontend` directory:
   ```sh
   cd frontend
   ```
2. Install dependencies (only needed the first time):
   ```sh
   npm install
   ```
3. Start the frontend app:
   ```sh
   npm start
   ```
4. The frontend will run on [http://localhost:3000](http://localhost:3000)

---

Make sure both apps are running for full functionality.
